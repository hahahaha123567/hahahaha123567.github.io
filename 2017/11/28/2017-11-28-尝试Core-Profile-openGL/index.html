挖个坑，放了假有时间配一下 glfw + glew 或 glfw + glad 的环境，感受一下现代openGL的编程过程

基本概念

OpenGL

OpenGL 只是一个规范，具体的实现由 各个库 的编写者自己决定

实现OpenGL功能的库

  • GLUT - The OpenGL Utility Toolkit
  • FreeGLUT - A free-software/open-source alternative to the (GLUT) library
  • SDL - A cross-platform development library designed to provide low level access to audio, keyboard, mouse, joystick, and graphics hardware via OpenGL and Direct3D
  • GLFW - Free, open source, portable framework for OpenGL application development

确定当前程序使用的OpenGL版本在驱动上的对应函数的库

  • GLEW - OpenGL Extension Wrangler Library
  • GLAD - Multi-Language GL/GLES/EGL/GLX/WGL Loader-Generator based on the official specs
